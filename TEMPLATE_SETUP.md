\# Niche SEO + Affiliate Template Playbook



This repo is tuned for \*\*Payroll for Crews\*\*, but the framework is reusable.



This file is a checklist for cloning the system into a new niche without

having to remember all the steps.



\## 0. Requirements



\- Node and npm installed

\- Git installed and authenticated with GitHub

\- OpenAI API key (stored in `.env` as `OPENAI\_API\_KEY=...`)

\- Google account for GA4 / Search Console

\- Netlify account



\## 1. Copy this repo for a new niche

&nbsp;  - `pajamas-and-pillows`

2\. On your machine:
